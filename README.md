# MuFabrika

MuFabrika application was coded using Swift and SwiftUI, AWS Dynamodb was used as database, scrum methodology was used in the development phase.


## Technologies
- Swift,
- SwiftUI,
- MVVM Artchitecture,
- AWS Dynamodb,
- SDWebImage,
- Clean Code,
- Scrum,
- QrScanner,
- GitLab,
- XCode
